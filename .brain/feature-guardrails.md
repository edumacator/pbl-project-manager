# Feature Guardrails

## Do not accidentally turn this into…
- A generic workplace PM suite.
- An LMS clone.
- A teacher surveillance system.
- An AI-first experience that replaces student thinking.
- A deeply nested, over-configurable admin platform.

## Guardrails
- Student workflows must stay simple and legible.
- Avoid duplicate or overlapping task systems.
- AI should scaffold thinking, not replace it.
- Teacher tools should support coaching and orchestration.
- Advanced features must not burden the default classroom workflow.
