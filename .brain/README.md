# .brain Operating Context

This `.brain` folder is the persistent operating context for AI coding agents and human contributors working in this repository.

## Why this folder exists
- Preserve the product intent of this high-school PBL project manager across sessions and tools.
- Reduce repeated prompting by storing durable operating guidance in one place.
- Prevent architecture and product drift during routine feature work and refactors.

Read these files before making meaningful changes.
