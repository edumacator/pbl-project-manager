# Workflow Rules

## Tool roles
- **ChatGPT project**: architecture direction, pedagogy alignment, feature tradeoffs, prompt design, implementation review.
- **Codex**: code edits, refactors, migrations, wiring, and tests.

## Process for larger changes
1. Inspect relevant code and docs.
2. Summarize understanding.
3. Propose plan.
4. Identify risks and constraints.
5. Implement.
6. Summarize changes and follow-ups.

## Process for small fixes
1. Inspect affected files.
2. Apply the minimal fix.
3. Summarize what changed.
