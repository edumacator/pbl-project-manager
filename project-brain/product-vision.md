# Product Vision

## What it Is
A lightweight project orchestration system for High School Project-Based Learning (PBL). It is a specialized tool that blends:
- **Project Structure**: Kanban tasks, milestones, and driving questions.
- **Student Scaffolds**: Cognitive tools to help students manage complexity.
- **Teacher Coaching Visibility**: Insights that help teachers identify "stuck" teams rather than just monitoring output.
- **Reflection & Critique**: Embedded peer feedback and self-reflection loops.

## What it Is NOT
- **A generic enterprise PM suite**: We don't need time-tracking, complex gantt charts, or resource billing.
- **A surveillance tool**: The goal is coaching support, not policing student minutiae.
- **A full LMS replacement**: We are not trying to be Canvas or Google Classroom. We do one thing well: PBL orchestration.
- **Bloated all-in-one platform**: Keep the feature set lean and focused on the PBL workflow.

## Primary Users
1. **Teachers**: Project designers and coaches. They need visibility into team health and the ability to set milestones.
2. **Students**: Contributors and reflective learners. They need help with organization and peer collaboration.

## Core Priority
**Student usability and educational value first, feature breadth second.**
